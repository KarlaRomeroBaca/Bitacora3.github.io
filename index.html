Frameworks

Un framework es un conjunto de herramientas, librerías, convenciones y buenas prácticas que pretenden encapsular las tareas repetitivas en módulos genéricos fácilmente reutilizables.

Los frameworks se utilizan en el ámbito de la programación de aplicaciones desde hace décadas.

Recientemente se han usado para programar y diseñar aplicaciones web, por lo que ya existen decenas de frameworks para CSS.

Un framework CSS es un conjunto de herramientas, hojas de estilos y buenas prácticas que permiten al diseñador web olvidarse de las tareas repetitivas para centrarse en los elementos únicos de cada diseño en los que puede aportar valor.

Unidades Responsivas

Las unidades responsivas son medidas que se adaptan al tamaño de la pantalla o contenedor en el que se despliega el contenido. Las principales unidades son:

% : Unidades porcentuales, dependen del tamaño del contenedor padre.

vw y vh: Vista Width y Vista Height, representan el ancho y alto del viewport, respectivamente. 1vw es igual al 1% del ancho de la ventana, y 1vh al 1% de su altura.

vmin y vmax: Basadas en la dimensión menor y mayor de la ventana, útiles para mantener escalas relativas sin importar la orientación de la pantalla.

em y rem: Basadas en el tamaño de fuente. em depende del tamaño del elemento padre, mientras que rem se basa en el tamaño de la fuente raíz.

Estas unidades ayudan a crear layouts que se adaptan dinámicamente a diferentes resoluciones y tamaños de pantalla.

Mobile First

Una investigación realizada por la GSMA (Asociacion Global de Sistemas de Comunicación Móvil) afirma que el uso de internet a tra vés de dispositivos móviles crecerá un 50% hasta 2020. Estar conectado en internet se volvió parte de nuestro día a día. Poco a poco estamos dejando de usar la computadora para realizar las tareas cotidianas y estamos usando cada vez más los smartphones.

En 2015, Google ya hablaba sobre esta tendencia, y priorizó los sitios web que son mobile friendly (sitios web amigables para dispositivos móviles) en los resultados de las búsquedas.

Los dispositivos móviles garantizan la practicidad que buscamos pero, para que la experiencia en internet sea buena, es necesario que la navegación sea pensada para estos dispositivos. Su propuesta era crear un sitio web pensado primero para dispositivos móviles y después ajustarlo para la computadora. El modelo Mobile First es una filosofía que perfecciona el proceso de diseño de cualquier sitio web, buscando mejorar el diseño web adaptable.

Esta nueva forma de diseñar, surge de los dispositivos móviles y de cualquier pantalla de tamaño menor, en la que se parte de una escala en aumento hasta adaptar el diseño a pantallas de gran tamaño.

Para aplicar este método, debemos tener en cuenta que todo lo que no sea esencial debe quedar fuera de nuestro diseño. Para hacerlo debemos pensar en lo que buscan nuestros consumidores, qué es lo que no puede faltar en ningón caso Y qué es prescindible o superfluo.

Teniendo en cuenta este enfoque de diseño, las dimensiones y capacidades de un móvil y la información que debemos incluir, podemos pasar al desarrollo de una página web.

Lo primero que se debe hacer es aclarar el contenido básico y separarlo del contenido sobrante o prescindible.

Modulo grid

El módulo CSS Grid es una herramienta para diseñar layouts en dos dimensiones: filas y columnas. Grid permite construir estructuras de diseño complejas y adaptables, organizando los elementos en una cuadrícula flexible. Esta estructura facilita la creación de diseños que se reorganizan y ajustan de manera automática según el tamaño de la pantalla. Grid es especialmente valioso en diseño adaptativo, pues permite cambiar la disposición de los elementos sin alterar su contenido.

Algunas características de CSS Grid incluyen la creación de columnas y filas de diversos tamaños, la posibilidad de definir áreas específicas dentro de la cuadrícula y controlar el espacio entre ellas. CSS Grid también permite definir un "flujo automático" de elementos, ideal para estructuras donde se requiere que los elementos cambien de posición o se reorganicen en función del tamaño del viewport.

@Supports

La regla @supports en CSS, también conocida como "Feature Queries" (Consultas de Funcionalidades), permite aplicar estilos condicionalmente, dependiendo de si el navegador admite ciertas propiedades o valores. Esto resulta útil para utilizar técnicas y propiedades de CSS modernas sin comprometer la compatibilidad en navegadores antiguos.

Por ejemplo, si se utiliza una nueva propiedad de CSS que algunos navegadores podrían no admitir, @supports permite especificar estilos alternativos o degradados para navegadores que no la soportan, asegurando una mejor experiencia para todos los usuarios. @supports también se puede combinar con otros operadores, como and, or, y not, permitiendo una lógica más compleja en la aplicación de estilos. Esto hace que @supports sea una herramienta clave en diseño adaptativo, pues permite implementar nuevas funcionalidades de CSS de forma segura.

Calc()

La función calc() es una herramienta que permite realizar cálculos matemáticos dentro de CSS. A diferencia de otras funciones que dependen de un valor absoluto o de un porcentaje predefinido, calc() permite combinar unidades y realizar cálculos complejos. Esto es muy útil cuando se requiere que el tamaño de un elemento sea el resultado de varias proporciones o unidades combinadas (como porcentajes, píxeles o rem), brindando así una mayor flexibilidad.

Por ejemplo, en diseños responsivos, calc() es útil para ajustar tamaños, márgenes y espacios, ya que permite especificar valores que cambian en función de otros elementos de la página o del viewport. La función admite operaciones básicas como suma, resta, multiplicación y división, y ofrece una solución flexible para lograr un diseño visual equilibrado y adaptable a diferentes tamaños de pantalla.

Cada una de estas herramientas de CSS —unidades responsivas, Grid, @supports, y calc()— contribuye a la creación de diseños adaptativos, asegurando que el contenido se muestre adecuadamente en una amplia gama de dispositivos.
